# Publisher

## Backend

### Admin Setup
- `cd backend/admin`
- Add environment variables in .env
- `docker compose up`

### lint
- `npm run lint`

### Format with Prettier
- `npm run format`


## Frontend

### Admin App
- `cd frontend/admin`
- add environment variables in .env
- `npm install`
- `npm start`


### Publisher App
- `cd frontend/publisher`
- add environment variables in .env
- `npm install`
- `npm start`
